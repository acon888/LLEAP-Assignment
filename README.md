# LLEAP-Assignment
This is my first Automation project use to test some functinalities of LLEAP application

## Required Softwares and Libraries:
1. The application uses python 3.14 and I havent tested with a different version lower than 3.14
2. Sikuli RobotFramework library
3. Pycharm IDE, VS Code can also be use in this application
4. Java version 8, but I havent tried using a different version of Java Runtime Environment
5. LLEAP application version 8.7.3
